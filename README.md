# Autonomous Exploration Robot

A research-oriented autonomous mobile robot developed from scratch using ROS 2. This project focuses on autonomous exploration, mapping, navigation, and semantic perception in simulation.

---

## Project Goals

- Design a differential-drive mobile robot
- Build the robot model using URDF/Xacro
- Simulate the robot in Gazebo
- Perform autonomous mapping with SLAM Toolbox
- Navigate using Nav2
- Detect objects with YOLO
- Build a semantic map of the environment

---

## Tech Stack

- ROS 2 Humble
- Gazebo
- RViz2
- URDF/Xacro
- SLAM Toolbox
- Nav2
- OpenCV
- YOLO
- C++
